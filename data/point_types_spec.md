# Point Types (Spec v2)
Each point has: `id`, `name`, `type`, `coords [lat, lng]`, `stay_duration_min`, `description`, `links`, `parking`, `price_estimate`.

Types used so far:
- `van_pickup`
- `park`
- `viewpoint`
- `camp`
- `city`
- `activity`
- `museum`

Pricing defaults (if missing):
- National Park entry: $35/vehicle
- City parking: $4/hour
- Public lot: $15/day
- RV hookup: $55/night
- Museum adult ticket: $25
